## 📌 Proje Hakkında

Bu proje, Python ile **keylogger** (tuş kaydedici) mantığını anlamak için geliştirilmiştir.  
Siber güvenlik uzmanlarının zararlı yazılımları anlaması ve kendini koruması için eğitim amaçlıdır.

### 🎯 Hedefler

- Klavye girişlerini yakalama mantığını öğrenmek
- Log dosyası oluşturma ve yönetme
- Olay tabanlı programlama
- Güvenlik farkındalığı kazanmak

---

## 🛠️ Özellikler

| Özellik | Açıklama |
|---------|----------|
| ⌨️ **Tuş Kaydı** | Tüm tuşları kaydeder (harf, sayı, özel tuşlar) |
| 📁 **Log Dosyası** | Tüm tuşları `logs/tus_kayitlari.txt` dosyasına kaydeder |
| ⏰ **Zaman Damgası** | Her satırın zamanını kaydeder |
| 🚪 **Güvenli Çıkış** | ESC tuşuna 3 kez basarak çıkış yapılır |
| 📊 **İstatistik** | Log dosyası analizi ve istatistik gösterimi |
| 🎨 **Renkli Çıktı** | Terminalde anlık tuş takibi |

---

## 📋 Taranan Tuşlar

| Tuş Türü | Örnekler |
|----------|----------|
| **Normal Tuşlar** | a, b, c, 1, 2, 3, !, @, # |
| **Özel Tuşlar** | [ENTER], [SHIFT], [CTRL], [ALT], [WIN] |
| **Yön Tuşları** | [UP], [DOWN], [LEFT], [RIGHT] |
| **Fonksiyon Tuşları** | [F1], [F2], [ESC] |
